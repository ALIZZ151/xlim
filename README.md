# XLIM STORE

Website store digital dengan tema **Ocean Blue Black**, dibuat untuk penjualan layanan panel hosting, VPS, private node, dan cloud server.  
Project ini memakai **Supabase** untuk database/auth, **Google Login** untuk customer area, dan **Vercel API** untuk menjaga akses admin tetap aman.
